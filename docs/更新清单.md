- 禁用代理层 Axum 默认 2MB body 限制，大上下文 LLM 请求不再被 413 截断
- 端点管理页/顶部导航隐藏滚动条且不影响其他页面
- 顶部导航横向列表支持纵向滚轮映射为左右滚动
- 隐藏密码输入框浏览器原生显隐按钮，避免与自定义眼睛图标重复

- 统计页历史记录弹窗表格溢出修复（覆盖默认 sm:max-w-lg 限制）
- 修复 codex 多轮历史切到 openai chat 端点时，function_call.arguments 残缺 JSON 原样透传导致上游报 `prefill failed: unexpected end of data`：转换层增加 JSON 完整性校验，合法规范化、残缺降级为 `{}` 并告警
- 修复 token 用量合计双重计算 cache_read：OpenAI 风格 input_tokens 已含 cached_tokens，统一归一化为净输入（`max(input - cache_read, 0)`），覆盖非流式、透传流式及 Chat→Responses/Claude 两条转换流式路径
