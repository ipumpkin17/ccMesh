# ccMesh 更新清单

## 新增功能

- 端点模型列表获取与可用性探测
- 启动行为设置：静默启动、开机自启、自动运行代理
- 端点模型点亮态（标记常用模型）
- 模型映射按点亮模型过滤候选
- 应用单例与桌面快捷唤起
- 端点卡片 API URL 点击跳转浏览器
- 端点类型筛选增加 Codex

## 修复与优化
- 修复mac图标边距导致图标偏大问题
- 应用代理端口不一致导致的失效问题
- 代理端口统一读取 `port` 配置键，保存后同步刷新状态
- Codex 默认 TOML 端口随设置动态更新
- 弹窗限高并支持内容滚动
- macOS Dock 图标点击恢复并聚焦主窗口
- `/v1/models` 跨端点模型去重，防止配置模型获取重复
