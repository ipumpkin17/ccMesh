# 端点同步字段完整性

## 问题

WebDAV 同步或本地导入后，端点的可用模型列表、模型映射经常丢失，需要重新拉取和配置。多密钥在部分恢复路径下也不完整。

## 决策

- 本地配置包 `EndpointExport` 补齐 `modelMappings`、`fast`、`fastSortOrder`
- 本地导入导出读写上述字段
- WebDAV 合并端点时同步 `endpoint_credentials`
- 导入/恢复成功后发出 `endpoints-changed`，前端即时刷新
- 创建/删除/克隆端点同样发出变更事件，保证同步链路一致

## 验收标准

- 配置导出再导入后，模型列表、点亮模型、映射、多密钥、快速队列标记保持一致
- WebDAV 覆盖恢复后，上述字段保持一致
- 恢复或导入后端点列表无需手动刷新
