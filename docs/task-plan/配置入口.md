目前打算主要实现Claude Code，Codex配置的抽取、存储、应用、覆盖原配置功能





抽取：某一些原配置文件的字段不动，应该保持原样，所以需要在工作开展之前，就应该每次都重新获取一些配置源的非操作字段，然后进行字段文件存储。为后续拼凑出网站配置做基础。

存储：用户在界面新增一个配置项，配置的表单，能和原配置的一些字段对应上。俗称最小字段写入，也叫操作字段

应用：存储的操作字段按文件的形式被读取出来，整合到一个最终展示完整配置信息的编辑器

覆盖原配置：抽取的非操作字段+用户编辑后的表单（操作字段）整合出来的完整配置信息，通过覆写形式。到原配置文件。







在一个表单和编辑器的页面，有配置名（渠道名）-一些表单项，和编辑器

用户操作流程：

1. 进入配置文件 页面
2. 切换tab，展示不同的配置文件界面（Cluade/codex）
3. 左侧布局是已保存的配置文列表（读取对应工作目录的渠道信息），通过右键可以实现删除，点击左侧布局顶部的操作区，有新增的图标按钮，
4. 中间没加载的时候是空图标占位,右侧是空编辑器，底部的操作区的按钮是灰色的
5. 点击添加后中间会加载一个空配置页面，有表单，有可以编辑的配置文件，表单顶部有一个tab选项：端点配置写入 ，自定义配置写入
6. ​	表单需要添加的是base_url ,model,就是操作字段的一些内容，后面具体说明和设计
   1. 不同点是：端点配置的base_url是读取的本地网关的地址：地址加端口，模型的话是选择能对外暴露的模型
   2. 自定义的话，就是可以自由配置，不现在输入的地址还有模型。
7. 中间表单下面还有一个编辑器，显示表单和操作字段的关系，根据填写的表单实时的显示响应字段，支持格式。默认支持编辑
8. 右侧是完整的整合的配置信息，右侧的编辑器顶部有一个开关，默认是关闭，开启后才能直接修改，支持点击格式化按钮，格式化。
9. 最后确认无误后，会看向底部的固定的操作区，点击应用，这是就提交信息，实现覆写。







Claude表单配置拆分： -  有两个表单，两个配置项，编辑器都支持格式化 布局读取（./claude部分配布局说明.png）

1. 模型表单项 -控制Sonnet  Opus Haiku 模型的填写 对应，每个模型后面有一个1m选择框，如果勾选就是mimo-v2-omni[1m]，没勾选所以是mimo-v2-omni
1. ANTHROPIC_DEFAULT_SONNET_MODEL，ANTHROPIC_DEFAULT_OPUS_MODEL，ANTHROPIC_DEFAULT_SONNET_MODEL
2. 地址信息对应ANTHROPIC_BASE_URL
2. 秘钥对应ANTHROPIC_API_KEY
3. 默认模型ANTHROPIC_MODEL



claude 环境的的原子写入是：

```
"ANTHROPIC_BASE_URL": "https://cc",
"ANTHROPIC_API_KEY": "sk-",
"ANTHROPIC_DEFAULT_HAIKU_MODEL":  "mimo-v2-omni"
"ANTHROPIC_DEFAULT_OPUS_MODEL":   "mimo-v2.5-pro[1m]"
"ANTHROPIC_DEFAULT_SONNET_MODEL": "mimo-v2.5-pro[1m]"
"ANTHROPIC_MODEL":                "mimo-v2.5-pro[1m]"
```



因为用户还可以添加自己的东西，所以最后存储的就是这些信息。



Codex表单配置拆分： -  布局读取（./codex部分配置布局.png）

1. 模型表单项 -控制默认模型，审核模型
2. model，review_model
3. 地址信息对应base_url
4. OPENAI_API_KEY 对应auth.json的秘钥值

其他没有配置的都是默认值，作为模板值

config.toml环境的操作 字段写入

```
model_provider = "OpenAI"
model = "gpt-5.5"
review_model = "gpt-5.5"
model_reasoning_effort = "high"
disable_response_storage = true
model_context_window = 1000000
model_auto_compact_token_limit = 900000

[model_providers.OpenAI]
requires_openai_auth = true
wire_api = "responses"
base_url = "http://127.0.0.1:3000/v1"
name = "OpenAI"
```
