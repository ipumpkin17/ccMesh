## 提交策略

按工作包/模块/目录分批提交，使用 git-commit-bot：
- WP1 后端迁移与解析、仓储、命令/事件、清理可分多次提交（迁移+模型一组、解析一组、明细表+采集一组、命令+事件一组）。
- WP2/WP4 前端按"组件/页面"提交；共享 `RequestMonitor` 单独成提交。
- WP3 后端读取器与命令一组、前端展示一组。
